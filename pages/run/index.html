    Run, Runner and Wheel¶

    You can run job using either the formatted form or a pseudo cli.

    Formatted¶

    For function completion and documentation, use the parse module setting. Custom modules should also be present.

    Target type¶

    Target¶

    This depends on the selected Target type.

    Function¶

    Function completion and documentation are taken from the parse module setting.

    Args and Keyword Arguments¶

    Use these fields for functions. If functions use named args or kwargs, they should be suggested.

    Schedule¶

    Schedule a recurring job, or postpone it.

    Test button¶

    The test button will run the selected function with test=True kwarg set.

    CLI¶

    This pseudo CLI should work like the salt command. Please refer to Pepper documentation.
